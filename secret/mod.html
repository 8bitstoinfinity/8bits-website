Discord Moderator Guide

These are guidelines for moderators on the 8 Bits to Infinity Discord server. As a moderator, you are expected to understand the powers and uphold the responsibilities outlined in this document.

Terms

Whenever "Josh" is mentioned in this guide, it means our community manager and jam host, Retro Indie Josh. You can ping or message him @RetroIndieJosh on Discord, tweet or DM on Twitter @RetroIndieJosh, or email seciden@gmail.com as a last resort.

By "moderator" we include those with the roles Admin or Moderator.

By "user" we mean any non-moderator on the server.

Moderator Levels

We have two moderator levels: Community Manager and Moderator. Josh is the only Community Manager.

Requirements

As a requirement for Discord community security, you must set up two-factor authentication (2FA) on your Discord account in order to use any moderator-level powers. This is a good idea for security, anyway.

Moderator Powers

As a moderator, you have the power to:

  1. Kick users from the server (removes them but they can rejoin)
  2. Ban users from the server (removes them and prevents them from rejoining)
  3. Change a user's display name on the server
  4. View and post in moderator-only channels
  5. Assign roles to and remove roles from users
  6. Add or remove server emojis
  7. Take priority on a voice channel with the Push to Talk (Priority) keybind

Your Responsibilities

Know and enforce the rules

Our rules are listed on the #welcome-and-rules channel. Your responsibilities are to:

  1. Know these rules and [our mission](http://8bitstoinfinity.com/index.php?page=Mission)
  2. Understand these rules (ask Josh or another moderator if you have any questions)
  3. Answer any user questions about these rules
  4. Enforce these rules with warnings, kicks, and bans

Inform Josh of any mistakes

If you spot a mistake or oversights in server documentation, channels, channel descriptions, etc., bring it to Josh's attention as soon as possible so he can fix it.

Encourage server discussion

When a user posts in a discussion channel, it's critical that they receive a response within a reasonable time frame. Take initiative to interact if you are able. This helps maintain a supportive atmosphere, and also feeds the Discord algorithm for things like server discovery listing.

Keep up-to-date with conversation in #power-core

The #power-core channel is where moderators and administrators can discuss happenings on the server. Josh often asks for assistance on this channel with any major server changes.

You can also discuss whether to warn, kick, or ban a user in this channel if Josh is not immediately responding to your concern

Inform Josh of upcoming absences

If you will be inactive in the server for an extended period of time (a month or more), please inform Josh and the rest of the team. You don't need to provide a reason, just an estimated return time. You are not required to be involved in the server 24/7 but it is important that everyone on the team is aware of extended absences.

Respond to @moderator pings

All moderators are notified when a user pings @moderator. Anyone on the server can ping this role. If you are unsure how to handle the situation, ping Josh.

Run bot commands in the #mod-bot channel

The #mod-bot channel is for moderators to run bot commands when the output would be spammy or reveal sensitive information. Please limit any warnings, kicks, bannings, or investigations into a user's history to this channel.

When in doubt of which channel to execute a bot command in, as a moderator, you should default to #mod-bot just in case.

Warn, kick, or ban users who break the rules

Although some leniency should be allowed for those who have simply misunderstood or failed to read the rules, repeated offenses should result in consequences.

Whenever you warn a user for breaking a rule, be sure to use the !warn command so we can track how many times they have been warned. Supply a message with the !warn command explaining *why* you are warning the user, for record-keeping.

For example, if Josh innocently posts a link to another Discord server because he didn't read the rules carefully, remind him to review the #rules channel, then use the command:

!warn @RetroIndieJosh Posted a link to a Discord server

Then, check in the #mod-bot channel to see if this user has previously broken this rule:

!warnings @RetroIndieJosh

With three warnings, kick the user, again with a message explaining why:

!kick @RetroIndieJosh Posting Discord links

With five or more, ban the user, and once more explain why:

!kick @RetroIndieJosh Posting Discord links other than 8bits

For more severe offenses such as clear hostility, hate speech, pornography, illegal material, or having a Discord link as their username, issue a ban outright. If you are unsure, contact Josh immediately.

Promote the Discord server and the 8 Bits to Infinity community in general

Moderators are meant to represent the best of 8 Bits to Infinity and what we have to offer, including our game jams, Discord server, website, game development resources, and more.

As a moderator, you are a spokesperson for our community and a protector of our values. Do your best to spread the word about our mission on social media and when interacting with our target demographic, game developers.

When possible, share any official content including but not limited to:

  • tweets by our Twitter bot (@8btiOfficial)
  • posts on the #announcements channel
  • the 8 Bits to Infinity newsletter
  • new and upcoming game jams and game development resources

Do not abuse your powers

Use your powers only to enforce the rules and keep the community a safe, welcoming place for all creative people.

We have a zero-tolerance policy for power abuse, even in a joking manner. If you are found to have abused a moderator power, you will be stripped of the moderator role. Depending on severity, you may also be kicked or banned from the server.
